# Airport Transfer Voucher Generator

Simple web app that generates a **clean airport transfer voucher PDF**.

Built with **HTML, CSS and Vanilla JavaScript**.
Runs completely in the browser — **no backend required**.

## Features

* Live voucher preview
* One-click PDF download
* Random booking number
* Optional return transfer
* Single-page A4 voucher layout

## Technologies

* HTML
* CSS
* JavaScript
* html2pdf.js

## Usage

1. Fill in the transfer details
2. Preview the voucher
3. Click **Download PDF**

The voucher will be generated and downloaded as a PDF.

## Deployment

Since this is a static app, it can be hosted easily on:

* GitHub Pages
* Cloudflare Pages
* Netlify
